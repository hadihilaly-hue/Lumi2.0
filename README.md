# Political Research Hub

A comprehensive web application for political science and current affairs research.

## How to Use

### Method 1: Direct File Opening
1. Locate the `index.html` file in this directory
2. Right-click on `index.html`
3. Select "Open With" and choose your web browser (Chrome, Firefox, Safari, etc.)
4. The app will open and work fully in your browser!

### Method 2: Using a Web Server (if Method 1 doesn't work)
1. Open a terminal in this directory
2. Run: `python3 -m http.server 8000`
3. Open your browser and go to: `http://localhost:8000`

## Features

- **News Aggregation & Search**: Search political topics and save articles to notes
- **Note-Taking & Annotations**: Create and organize research notes with tags
- **Source Management**: Add sources and generate citations (APA/MLA/Chicago)
- **Data Analytics**: Visualize your research with interactive charts

## Tips

- All data is saved automatically in your browser's localStorage
- Use tags in your notes to organize topics
- Click "Save to Notes" on search results to quickly capture information
- Export citations by clicking the "Copy Citation" button

Enjoy your research!
